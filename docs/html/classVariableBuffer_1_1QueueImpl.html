Detailed Description

The QueueImpl class implements the buffering strategy for a single logged variable.

This consists of:

  • a pool of available blocks, accessed only by the producer thread
  • a "read queue": produces pushes ready-to-consume blocks into it
  • a "write queue": consumed blocks are pushed into the queue in and finally return inside the pool
